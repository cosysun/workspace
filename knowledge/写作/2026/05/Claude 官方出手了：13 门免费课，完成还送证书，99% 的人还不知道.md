## 这 13 门课，到底讲什么？

我把它们按人群分成四条学习路径，你对号入座：

### 🟢 路径一：普通人 / 职场人（先从这里开始）

**① Claude 入门指南**

https://anthropic.skilljar.com/claude-101

Claude 怎么用、用在哪、有哪些核心功能。完全零基础，适合第一次接触 AI 工具的人。

**② AI 通识基础课**

https://anthropic.skilljar.com/ai-fluency-framework-foundations

不讲工具，讲思维。什么是 AI、怎么和 AI 协作、如何在工作中真正用起来。入门必学。

### 🔵 路径二：开发者 / 技术人

**③ Claude API 全解析**

https://anthropic.skilljar.com/claude-with-the-anthropic-api

函数调用、工具使用、流式响应，全部实战讲解。Python 基础就能上手。

**④ Claude Code 实战**

https://anthropic.skilljar.com/claude-code-in-action

把 AI 深度集成进开发工作流，包括 GitHub 集成、自定义命令、上下文管理，15 节课。

**⑤ 智能体技能开发**

https://anthropic.skilljar.com/introduction-to-agent-skills

在 Claude Code 中配置可复用指令集（Hooks），让 AI 记住你的工作习惯。

**⑥ MCP 入门：用 Python 搭建 MCP 服务端**

https://anthropic.skilljar.com/introduction-to-model-context-protocol

MCP（模型上下文协议）是今年最火的 AI 开发方向，这门课从零开始，1 小时入门。

**⑦ MCP 进阶：生产级部署**

https://anthropic.skilljar.com/model-context-protocol-advanced-topics

采样机制、文件系统操作、生产环境部署，给已经入门 MCP 的人继续深挖。

## 为什么 Anthropic 要免费开放这些？

说白了，这是一场 **生态战** 。

OpenAI 在卖 ChatGPT Plus，Google 在推 Gemini Advanced，而 Anthropic 选择了另一条路： **把开发者和用户先教会，再绑定生态** 。

你学完 MCP，第一反应是去用 Claude 的 API；你学完 Claude Code，工作流就自然接入了 Anthropic 的体系。免费课程是最高性价比的用户获取方式。

对我们来说，这是 **薅羊毛的好机会** 。
