# 强烈推荐：2026 年 AI 工程书单

![2026 AI 工程书单速查：12 本分四类](https://ai-content-1300152858.cos.ap-shanghai.myqcloud.com/writing/2026/06/2026-06-02-12-ai-books-cn/cover.png)

前阵子在 Medium 看到 Anubhav Goyal 整理的 2026 AI 工程书单，一共 12 本，感觉挺好的，分享给大家。

## 快速路径

- **后端工程师转 AI**：*AI Engineering*（Chip Huyen）矫正心智 → *Generative AI Design Patterns*（Lakshmanan & Hapke）把架构经验接进来
- **设计 Agent**：*Designing Multi-Agent Systems*（Victor Dibia）理解第一性原理 → *Agentic AI Engineering*（Yi Zhou）防止 agent 在生产里出事
- **做 RAG**：*Mastering RAG*（Ranajoy Bose）管检索 + *System Design for LLMs*（Marc Rolland）管生成
- **工程负责人**：上线前先读 *LLMOps*（Abi Aryan），避坑，免得账单咋了

## Foundation：先把心智模型摆正

**1. AI Engineering — Chip Huyen**
帮你从"训练模型"切换到"调用基础模型"的心智。读完你会把评估和数据集工程当成头等事。中文版已由宝玉译出（人民邮电·图灵，2026/02）。

**2. Hands-On Large Language Models — Jay Alammar & Maarten Grootendorst**
整条 LLM 流水线：tokenization、positional embedding、attention、生成参数都讲一遍。看完不会再把 LLM 当黑盒——输出垃圾的时候，能定位到是哪一段坏了。

**3. LLM Engineer's Handbook — Paul Iusztin & Maxime Labonne**
一本纯工程书。从数据采集到部署，跟着跑一个开源 LLM Twin 的全流程。读完你敢直接从 Hugging Face 拉模型自己 fine-tune，而不只是缩在闭源 API 后面。

## Agents：超越"贴个框架就跑"

**4. Designing Multi-Agent Systems — Victor Dibia**
作者来自微软 AutoGen Studio。这本书让你从零写一个 agent 库，而不是教你怎么用现成框架。读完你能讲清楚 LangGraph、AutoGen 为什么设计成这样，也明白一个 agent 怎么被人类打断、上下文怎么改、流程怎么再续上。

**5. AI Agents in Action — Micheal Lanham**
讲 agent 怎么接真实工具、怎么用 Model Context Protocol、怎么做检索增强记忆、怎么在容器化沙箱里跑。看完不会再把 agent 当本地玩具——它真要落地，得有一个能在云上安全跑代码的环境。

**6. Building Agentic AI — Sinan Ozdemir**
盯住企业 agent 在成本、速度、精度、隐私这四个角上的权衡。讲 quantization、speculative decoding 这类压低延迟的工程手段。读完你心里会有一套 precision / recall / latency 的评估框架。

**7. Agentic AI Engineering — Yi Zhou**
讲合规行业里的 agent 怎么活下来。书里提出 Agentic Stack 和 Trust Envelope——给 agent 套一个执行围栏，只能在授权范围里动手。读完你不会再骂模型乱来，会先去把权限和审计补好。

## Production & Ops：账单和故障的真实世界

**8. LLMOps — Abi Aryan**
传统 MLOps 在生成式 AI 面前几乎要重写一遍。书里讲 prompt drift（比如：API 厂商悄悄换了权重，两个月前还好好的 prompt，突然就坏了）、新一代监控、自动回归测试。读完不会再拍脑袋发上线。

**9. AI Systems Performance Engineering — Chris Fregly**
全书最硬核的一本：GPU 显存、CUDA kernel、KV cache 碎片，全讲。讲 hardware/software co-design 怎么把吞吐压到极限。看完不会一遇延迟就堆更贵的卡，而会先做 profiling。

**10. Generative AI Design Patterns — Valliappa Lakshmanan & Hannes Hapke**
32 个模式覆盖幻觉、不确定性、上下文越界这类常见架构问题。每个模式都给出问题定义、解法代码和 tradeoff。

## RAG & Safety：让生成步骤别再翻车

**11. Mastering Retrieval-Augmented Generation — Ranajoy Bose**
把 RAG 从周末 prototype 推到企业生产：文档分块、graph RAG、embedding 微调、dense + sparse 混合检索全讲透。看完你不会再死磕一条纯向量相似度。

**12. System Design for Large Language Models — Marc Rolland**
把 prompt 当系统边界来设计，而不是当文案来抠。从 systems engineering、安全分析、控制论里借方法，搭出 instruction hierarchy 和 task decomposition。看完你不会再在系统提示词里抠形容词，会去搭可观测性。

## 怎么挑

12 本不要全读, 大家根据自己的领域，先挑三本看看。
